For HTML parsing, you could add parse rules, either in your schema or in a custom `clipboardParser` 121 prop. For text, you probably want the \[`clipboardTextParser`\]([http://prosemirror.net/docs/ref/#view.EditorProps.clipboardTextParser 163](http://prosemirror.net/docs/ref/#view.EditorProps.clipboardTextParser)\] prop, which allows you to give a function from the pasted string to a document slice.
